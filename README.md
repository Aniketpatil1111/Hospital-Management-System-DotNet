# Hospital Management System (.NET)
A web-based Hospital Management System developed using ASP.NET and SQL Server to manage patients, appointments, and billing.
## Tech Stack
- C#
- ASP.NET
- SQL Server
- HTML, CSS, JavaScript
## Features
- Patient registration and management
- Appointment scheduling
- Billing and invoice generation
- CRUD operations using SQL Server
